Configuração

Configure a URL da sua API Vercel para ativar dados automáticos

URL da API (Vercel)
Exemplo: https://testelab.vercel.app
Passo a passo de configuração
1
Acesse github.com, crie um repositório chamado testelab como Public
2
Dentro do repositório, crie api/summary.js e api/campaigns.js com o conteúdo que o Claude forneceu
3
Acesse vercel.com → login com GitHub → Add New Project → importe o repositório testelab
4
Adicione a variável de ambiente: Name: UTM_TOKEN → Value: seu token UTMify → clique em Deploy
5
Copie a URL gerada (ex: testelab.vercel.app), cole acima e clique em Testar conexão
